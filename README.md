# HL2RP

HL2RP is a roleplay schema for [Lilia](https://github.com/LiliaFramework/Lilia).

# Contents

- Combine Locks
  
- Flashlights
  
- Combine Forcefields
  
- Paper Notes

- Text Radios
  
- Combine Scanners

- Shootable Locks

- Tying

- Tying Search

- Text to Speech Voices
